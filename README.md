# Predicting Monthly MRSA Hospital Infections Using a Compartmental SCIR Transmission Model


### Description
This study aims to evaluate a compartmental transmission model proposed by Hogea et al., which explicitly incorporates colonization, to assess its ability to capture MRSA infection trends in hospitals. Using surveillance data from 2004 to 2008, we estimate key transmission parameters and aim to improve forecasting for MRSA outbreaks and inform infection control strategies. This model was chosen because few studies have explored such parameters, and it offers a more realistic approach by including colonization as a critical factor in MRSA transmission.

The proposed Susceptible-Colonized-Infected-Recovered-Susceptible (SCIR) model is governed by a system of ordinary differential equations (ODEs), where transitions between compartments are determined by biological and epidemiological
processes. The original model included both MRSA and MSSA infections, however I have adjusted the model slightly to account for only MRSA infections which are the focus of this project.

### Methods
Surveillance data from a hospital system in Maricopa County, Arizona, spanning from 2004 to 2008, was used for model fitting and evaluation. The dataset was split into a train-ing period (2004–2006) and a testing period (2007–2008). The training dataset, consisting of 29,385 observations, was used for parameter estimation while the testing dataset, consisting of 29,418 observations, served for model validation. Monthly infection counts for MRSA were aggregated, and the data was converted into a time series of observed infections for both periods.

The SCIR compartmental model was employed to capture the dynamics of MRSA transmission. The model's parameters were estimated using a Metropolis-Hastings Markov Chain Monte Carlo (MCMC) method. To obtain parameter estimates, a Poisson likelihood function was used, where observed infection counts were compared with the model's predictions. Initial parameter estimates were specified using values found in the literature. New parameter sets were proposed and accepted or rejected based on the likelihood ratio between the proposed and current sets. This process was repeated for 10,000 iterations to explore the parameter space.

Model simulations were carried out using the ode() function from the “deSolve” package in R, which numerically solved the system of differential equations over time. Once the parameters were estimated, posterior predictive distributions for infection counts were generated by running the model with 500 posterior samples. These simulations produced predicted infection counts for each month during the testing period. The model's performance was evaluated by comparing the predicted infection counts with the observed infections in the test dataset, al-lowing for the assessment of the model’s accuracy in forecasting MRSA outbreaks. All data preparation and analyses were conducted using R version 4.4.1.
